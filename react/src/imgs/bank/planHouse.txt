ПАНЕЛЬ У ДОМА
Игрок встает на колшейп у дома, вызываю у тебя событие открытия панели с информацией и передаю информацию
СОБЫТИЯ(в браузере)
+1)'house.menu'() - открыть панель информации
+2) 'house.load(houseInfo)' - передать информацию
СВОБОДЕН
	houseInfo = {
            player.call('enterHouseMenuAns.client',[{
            name: houses[currentHouse].name,
            class: houses[currentHouse].class,
            numRooms: houses[currentHouse].numRooms,
            garage: houses[currentHouse].garage,
            carPlaces: houses[currentHouse].carPlaces,
            rent: houses[currentHouse].rent,
            price: houses[currentHouse].price,
            pos: houses[currentHouse].pickup
        }
	КУПЛЕН
	houseInfo = {
            name: houses[currentHouse].name,
            class: houses[currentHouse].class,
            numRooms: houses[currentHouse].numRooms,
            garage: houses[currentHouse].garage,
            carPlaces: houses[currentHouse].carPlaces,
            rent: houses[currentHouse].rent,
            owner: houses[currentHouse].owner.nick,
            pos: houses[currentHouse].pickup
        }
+3)'house.menu.close'() - закрыть панель информации
+4)'house.menu.enter'(place) - открыть меню входа 
	place = 0 - с улицы(войти в дом/гараж); 1 - из дома; 2 - из гаража
(после нажатия любой кнопки на меню входа начинает крутиться колесико загрузки до тех пор, пока оно не будет закрыто)
+5)'house.menu.enter.close'() - закрыть меню входа
6)'house.buy.ans'(ans, houseInfo) - результат покупки дома 
	ans = 0 не успешно; 1 успешно
	houseInfo = то же что и выше
+7)'house.enter.ans.err'() - ошибка входа куда-либо(писать что то типа "Дверь заперта")
(в случае, если нет гаража не будет вызываться меню house.menu.enter)
После нажатия любой кнопки на панели house.menu там начинает крутиться колесико загрузки до тех пор пока не придет ответ, house.enter.ans если была нажата клавиша войти/осмотреть или house.buy.ans если была совершена покупка купить дом


ТРИГГЕРЫ(на клиенте)
1)'house.buy'() - купить дом (спрашивать подтверждение покупки, типа он на кнопку нажал, а ты такой:"Точно хотите купить?")
+2)'house.enter'(place) - войти в(выбор варианта в меню входа(события 3), после чего закрыть меню входа)
	place = 0 - на улицу; 1 - в дом; 2 - в гараж
+3)'house.menu.close'() - панель информации была закрыта с панели
+4)'house.menu.enter.close'() - панель входа была закрыта с панели



ПЛАН РАБОТЫ:
ВХОД
куплен / не куплен
1) Игрок встал на колшейп ( EVENT: 'house.load', 'house.menu')
2) Открылась панель дома 
3) Выбрал войти 
4) Открылась панель куда(дом/гараж)
5) Выбрал нужный вариант ( TRIGGER: 'house.enter')
6) Колесико загрузки
7) Приходит ответ ( EVENT: 'house.enter.ans')
8) Если не успешно, показать ошибку, если успешно закрыть панель

ВЫХОД 
1) Игрок встал на колшейп ( EVENT: 'house.menu.enter') 
2) Открылась панель куда 
3) Выбрал нужный вариант ( TRIGGER: 'house.enter') 
4) Колесико загрузки 
5) Приходит ответ ( EVENT: 'house.enter.ans') 
6) Если не успешно, показать ошибку, если успешно закрыть панель

Если на сошел с колшейпа
ВХОД(вне зависимости выбрал уже осмотреть/войти или нет)
EVENT: 'house.menu.close'
ВЫХОД
EVENT: 'house.menu.enter.close'

Если была закрыта панель информации
TRIGGER: 'house.menu.close'
Если была закрыта панель входа
TRIGGER: 'house.menu.enter.close'
